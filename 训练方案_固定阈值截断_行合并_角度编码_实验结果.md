# 训练实现方案与实验结果

---

## 一、数据预处理

### 1.1 原始数据
- 数据文件: `mnist_train_1000_16_16.npz`
- 形状: (1000, 16, 16) - 1000个16×16灰度图像
- 像素值范围: 0.0 ~ 1.0

### 1.2 固定阈值截断（二值化）

对每个像素进行二值化处理：

```
if 像素值 > 0:
    像素值 = 1
else:
    像素值 = 0
```

**公式**: $x_{binary}(i,j) = \mathbb{1}_{x(i,j) > 0}$

### 1.3 行合并压缩

将16行图像两两合并，每2行所有元素求和，得到8个整数值：

```
合并方式:
第1行 + 第2行 → 1个值 (范围0~16)
第3行 + 第4行 → 1个值
...
第15行 + 第16行 → 1个值

结果: 16×16 → 8个整数值
```

**公式**: $x_{compressed}(k) = \sum_{i=2k+1}^{2k+2} \sum_{j=0}^{15} x_{binary}(i,j)$, 其中 $k=0,1,...,7$

## 二、量子编码

### 2.1 角度编码

将8个整数值编码到8个量子比特上：

```
编码公式: angle = (π/2) × value

示例:
value = 0  → angle = 0      (不旋转)
value = 4  → angle = π/2    (旋转90°)
value = 8  → angle = π      (旋转180°)
value = 16 → angle = 2π    (旋转360°)
```

**公式**: $\theta_i = \frac{\pi}{2} \cdot x_{compressed}(i)$, 其中 $i=0,1,...,7$

对每个量子比特应用RY旋转门：$RY(\theta_i)|0\rangle_i$

### 2.2 量子线路结构

```
【编码层】8个RY门（角度编码）
    ↓
【Ansatz层】RealAmplitudes × 5层
    ├─ 第1层: 8个RY(参数) + 4个CNOT
    ├─ 第2层: 8个RY(参数) + 4个CNOT
    ├─ 第3层: 8个RY(参数) + 4个CNOT
    ├─ 第4层: 8个RY(参数) + 4个CNOT
    └─ 第5层: 8个RY(参数) + 4个CNOT
    ↓
【测量层】测量量子比特0
```

**可训练参数**: 8量子比特 × 5层 = 40个

---

## 三、RealAmplitudes Ansatz

### 3.1 每层结构

```
for layer in range(5):
    # RY层
    for qubit in range(8):
        应用RY(qubit, params[layer*8 + qubit])

    # CNOT纠缠层（交替配对）
    if layer % 2 == 0:  # 偶数层
        for i in [0, 2, 4, 6]:
            应用CNOT(i, i+1)
    else:  # 奇数层
        for i in [1, 3, 5, 7]:
            应用CNOT(i, i+1)
```

### 3.2 参数数量

- RY门: 8个/层 × 5层 = 40个参数
- CNOT门: 4个/层 × 5层 = 20个（无参数）

## 四、测量与输出

### 4.1 测量

测量量子比特0，获取基态概率：

$$P(|0\rangle) = \sum_{x: x_0=0} |\langle x|\psi\rangle|^2$$

其中 $|\psi\rangle$ 是量子线路输出的末态。

### 4.2 分类概率

- $P(|0\rangle) \rightarrow 1$: 预测为数字4（类别0）
- $P(|0\rangle) \rightarrow 0$: 预测为数字9（类别1）

---

## 五、损失函数

### 5.1 绝对值损失

$$\mathcal{L} = |y - P|$$

其中:
- $y$: 真实标签 (0=数字4, 1=数字9)
- $P$: 量子电路输出的 $P(|0\rangle)$

### 5.2 损失设计

| 标签 | 真实标签y | P(|0⟩)含义 | 损失 | 优化目标 |
|------|-----------|------------|------|---------|
| 数字4 | 0 | P(数字4)应→1 | O = \|0 - P\| | P→1 |
| 数字9 | 1 | P(数字4)应→0 | O = \|1 - P\| | P→0 |

---

## 六、梯度计算

### 6.1 参数移位法则

对于参数 $\theta_i$，通过在 $\theta_i \pm \pi/2$ 处运行电路来近似梯度：

$$\frac{\partial \mathcal{L}}{\partial \theta_i} = \frac{\partial \mathcal{L}}{\partial P} \cdot \frac{\partial P}{\partial \theta_i}$$

其中:
- $\frac{\partial \mathcal{L}}{\partial P} = \text{sign}(P - y)$ (链式法则符号项)
- $\frac{\partial P}{\partial \theta_i} \approx \frac{P(\theta_i+\pi/2) - P(\theta_i-\pi/2)}{2}$

### 6.2 梯度计算步骤

```
对每个参数θi:
    1. θ+ = θi + π/2
    2. 运行电路得到 P+
    3. θ- = θi - π/2
    4. 运行电路得到 P-
    5. grad = sign(P-y) × (P+ - P-) / 2
```

**注意**: 每个样本需要计算40次前向传播（40参数 × 2次偏移）

---

## 七、参数更新

### 7.1 ADAM优化器

使用ADAM优化器更新参数：

```
m = β1 × m + (1-β1) × grad          # 一阶矩估计（动量）
v = β2 × v + (1-β2) × grad²         # 二阶矩估计
m_hat = m / (1 - β1^t)               # 偏差校正
v_hat = v / (1 - β2^t)               # 偏差校正
θ = θ - α × m_hat / (√v_hat + ε)    # 参数更新
```

### 7.2 超参数

| 参数 | 符号 | 值 |
|------|------|-----|
| 学习率 | α | 0.001 |
| 一阶矩衰减率 | β₁ | 0.9 |
| 二阶矩衰减率 | β₂ | 0.999 |
| 数值稳定项 | ε | 10⁻⁸ |

---

## 八、训练流程

```
for epoch in 10轮:
    打乱训练数据
    
    for batch in 20个批次 (每批50样本):
        对每个样本:
            1. 前传: 编码 → Ansatz → P(|0⟩)
            2. 计算损失: |y - P|
            3. 参数移位梯度: 40参数 × 2次运行
        平均梯度
        ADAM更新参数
    
    记录本轮平均损失
```

### 9.3 结论

- 损失呈持续下降趋势，模型正在学习
- 下降速度较慢，可能原因：
  1. 学习率较小（0.001）
  2. 数据预处理信息有损失（256维→8维）
  3. 角度编码表达能力有限
