# 本源杯量子机器学习竞赛 - 实验结果报告

## 一、实验概述

### 1.1 任务描述

- **任务**：MNIST 手写数字 4 和 9 二分类
- **框架**：UnifiedQuantum (uniqc)
- **目标**：使用量子机器学习方法对 MNIST 数据集进行分类

### 1.2 约束条件

| 参数 | 限制 |
|------|------|
| 量子比特数 | ≤ 8 |
| 可训练参数 | ≤ 100 |

### 1.3 评分公式

$$Score = 44 \times Acc + 3 \times \left[ \frac{8-Q}{7} + \frac{100-P}{99} \right]$$

其中：
- $Acc$：测试集准确率
- $Q$：量子比特数
- $P$：可训练参数数量

---

## 二、实验方法

### 2.1 量子电路设计

采用**量子特征提取 + 经典分类器**的混合方案：

```
输入数据 (256维)
        ↓
【振幅编码】将256维数据编码到8量子比特
        ↓
【Ansatz层】5层 RealAmplitudes（固定参数）
        ↓
【测量层】测量所有量子比特
        ↓
【特征提取】提取量子态概率分布作为特征
        ↓
【经典分类器】MLP神经网络
        ↓
输出：二分类预测
```

### 2.2 噪声模型

使用 uniqc 的 `DummyAdapter` 配置噪声：

```python
noise_model = {
    'depol_1q': 0.0,       # 单比特门无噪声
    'depol_2q': 0.0154,    # 双比特门去极化噪声
    'readout': 0.0698,      # 读取错误
}
```
