# 训练代码实现方案

## 一、数据处理

### 1.1 数据加载
```python
# 加载数据
x_train: (1000, 16, 16)
y_train: (1000,)  # 标签 0 或 1
```

### 1.2 数据预处理
```python
# 1. 展平为256维
x_train = x_train.reshape(1000, 256)

# 2. L2归一化
x_train = x_train / ||x_train||
```

## 二、量子线路结构

### 2.1 振幅编码（256维 → 8量子比特）
```python
# 将256维数据分成8组，每组32维
for i in range(8):
    chunk = data[i*32 : (i+1)*32]
    # 用RY门编码第一个分量
    angle = arcsin(chunk[0] / ||chunk||)
    circuit.ry(i, angle)
```

### 2.2 RealAmplitudes ansatz（5层，40参数）
```python
for layer in range(5):
    # RY门（可训练参数）
    for qubit in range(8):
        circuit.ry(qubit, params[layer*8 + qubit])
    
    # CNOT纠缠
    for i in [0,2,4,6]:
        circuit.cnot(i, i+1)
```

### 2.3 测量
```python
# 测量第0个量子比特
circuit.measure(0)
```

---

## 三、保真度计算

### 3.1 运行量子线路
```python
simulator = Simulator(backend_type="statevector")
probs = simulator.simulate_pmeasure(circuit)
# 返回：P(|0⟩), P(|1⟩), ...
```

### 3.2 提取P(|0⟩)
```python
# 统计测量结果中第0个比特为0的概率
F = P(|0⟩)  # 作为保真度
```

---

## 四、损失函数

### 4.1 标签感知损失
```python
# 数字4（y=0）：损失 = |0 - F| = F，希望F→1
# 数字9（y=1）：损失 = |1 - F| = 1-F，希望F→0
loss = abs(y - F)
```

### 4.2 批量损失
```python
# 对批量样本求平均
loss = mean(|y - F|)
```

---

## 五、梯度计算

### 5.1 参数移位法则
```python
# 对每个参数θi：
# 正向偏移：θ+ = θi + π/2 → F+
# 负向偏移：θ- = θi - π/2 → F-
# 梯度：grad = (F+ - F-) / 2
```

### 5.2 批量梯度
```python
# 对批量样本求平均梯度
grad = mean(grad_i)
```

---

## 六、参数更新

### 6.1 ADAM优化器
```python
m = β1 * m + (1-β1) * grad  # 动量
v = β2 * v + (1-β2) * grad²  # 方差
m_hat = m / (1-β1^t)
v_hat = v / (1-β2^t)
params = params - α * m_hat / (√v_hat + ε)
```
